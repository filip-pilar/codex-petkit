# Edit anonymous semantic review

Every edited V2 build receives three fresh anonymous semantic verdicts. Reviewers see randomized clip tokens, candidate state meanings, full-size and approximate UI-size filmstrips/GIFs, and calibration controls covering inert, repetitive, cropped/malformed, and identity/material-drift failures. They do not see row labels, prompts, motion plans, prior verdicts, or the private answer key.

Each reviewer maps every clip at both sizes, records the strongest alternative and visible evidence, rejects every calibration control, and compares idle/waiting, idle/work, idle/review, waiting/work, waiting/review, work/review, review/failed, and failed/idle. Use a distinct reviewer identifier in each verdict. All three verdicts must pass independently; a majority cannot rescue a semantic failure. If a state or pair fails, repair the complete row or return to semantic design.
