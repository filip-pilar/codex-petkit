# Artwork and generated assets

The Apache License 2.0 in this repository covers the source code,
documentation, skills, and synthetic test fixtures unless a file says
otherwise. It does not automatically license artwork supplied by users or
generated inside a local pet project.

The `pets/` directory is ignored because it may contain personal reference
images, generated artwork, immutable builds, private review answer keys,
installation backups, and machine-local history. Contributors must confirm
that they have permission to redistribute an asset before adding it to a
public example. Any public example must carry its own explicit asset license
and provenance note.
