# V2-only compatibility boundary

The production, import, validation defaults, project schema, build, acceptance, and installation paths support only `spriteVersionNumber: 2` and a 1536×2288 8×11 atlas. Contract version 1 deliberately fails to load.

The one-way `upgrade-project` command exists only to preserve the nine standard source rows, alpha geometry, and immutable history of pets created locally before look directions were added. The required completed-atlas despill may clean contaminated RGB under translucent edge pixels and records that fact in the migration change report. It does not restore a legacy build or package path.
